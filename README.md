# Car-show-case
Single-page automotive showcase built with HTML, CSS, and modular vanilla JavaScript. It features a BMW M4 Competition (2024) in a premium, responsive design. The app uses reusable modules for the navbar, car display, and footer.


This project is a **Single-Page Automotive Showcase** built using HTML, CSS, and modular Vanilla JavaScript that mimics a component-based architecture like React. 

- **Subject:** It currently showcases a **BMW M4 Competition** (2024 model, 503hp twin-turbo).
- **Architecture:** The application is split into reusable JavaScript modules (`Navbar.js`, `CarDisplay.js`, and `Footer.js`) which are injected into specific mount points inside the main `index.html` file.
- **Design:** It features a premium, responsive design tailored for a high-end feel, utilizing custom typography ("Outfit" from Google Fonts) and structured CSS styling to highlight the car's features. 

In short, it's a clean, modular front-end web application designed to present a premium car model to potential buyers with a modern, high-quality user interface!
